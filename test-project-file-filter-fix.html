🔧 Project File Filter Fix

🐛 问题

Project Manager的Open Project功能无法打开 *.prj.GAI 文件，显示为灰色。

✅ 修复

Electron文件过滤器不支持包含点的复合扩展名

❌ 修复前

extensions: ['prj.GAI']

✅ 修复后

extensions: ['GAI']

🧪 测试方法

1. 测试Project Manager: 打开Project Manager → 点击Open Project → 验证.prj.GAI文件可选择
2. 测试主菜单: Project → Open Project → 验证文件过滤器正确

🎯 预期结果

✅ 文件过滤器: "Genome AI Studio Project Files (*.prj.GAI)"

✅ .prj.GAI文件可正常选择

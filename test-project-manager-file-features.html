📁 Project Manager File Features Guide

Step-by-step guide to find and use the new file management features

⚠️ 重要提示： 如果你看不到文件管理功能，请按照以下步骤操作。可能需要创建测试项目来查看功能。

🚀 第一步：打开Project Manager并创建测试项目

创建测试项目以查看功能

  1. 打开Project Manager窗口
  2. 点击 🧪 Create Test Project 按钮
  3. 这将创建一个包含示例文件的项目

👁️ 第二步：文件预览功能

如何预览文件

  • 双击文件 - 直接双击任何文本文件（.fasta, .gff, .vcf, .txt等）
  • 悬停按钮 - 鼠标悬停在文件卡片上，点击 👁️ 按钮
  • 右键菜单 - 右键点击文件，选择 "Preview File"
支持的文件类型： FASTA (.fasta, .fa), GFF (.gff, .gtf), VCF (.vcf), BED (.bed), TXT (.txt), JSON (.json), XML (.xml)

✏️ 第三步：文件重命名功能

如何重命名文件

  • 悬停按钮 - 鼠标悬停在文件卡片上，点击 ✏️ 按钮
  • 右键菜单 - 右键点击文件，选择 "Rename File"
  • 键盘快捷键 - 选择文件后按 F2 键
注意： 系统会自动检查文件名冲突，防止重名文件

🗑️ 第四步：文件删除功能

如何删除文件

  • 悬停按钮 - 鼠标悬停在文件卡片上，点击 🗑️ 按钮
  • 右键菜单 - 右键点击文件，选择 "Delete File"
  • 多选删除 - 选择多个文件，按 Delete 键或使用菜单
安全提示： 删除前会有确认对话框，删除操作无法撤销

🎯 第五步：右键上下文菜单

完整的右键菜单选项

右键点击任何文件，应该看到以下菜单：

👁️ Preview File
✏️ Edit File (仅限文本文件)
📝 Rename File
⬇️ Download File
――――――――――
📋 Duplicate File
📁 Move to Folder
――――――――――
🗑️ Delete File
如果右键菜单不显示：
  • 确保你已经创建了测试项目
  • 确保文件列表中有文件
  • 尝试刷新页面（F5）
  • 检查浏览器控制台是否有错误

📋 第六步：其他文件操作

文件复制和移动

  • 复制文件 - 右键菜单 → "Duplicate File"
  • 移动到文件夹 - 右键菜单 → "Move to Folder"
  • 下载文件 - 右键菜单 → "Download File"

文本文件编辑

  • 预览文本文件后，点击 "✏️ Edit" 按钮
  • 编辑内容后点击 "Save Changes"
  • 支持的格式：FASTA, GFF, VCF, BED, TXT等

🛠️ 故障排除

如果功能不工作...

  1. 检查项目是否创建 - 确保你已经创建了测试项目
  2. 检查文件列表 - 确保项目中有文件显示
  3. 刷新页面 - 按 F5 刷新Project Manager
  4. 检查控制台 - 打开开发者工具查看错误信息
  5. 重新打开 - 关闭并重新打开Project Manager窗口

⌨️ 键盘快捷键

快捷键列表

文件操作：
F2 - 重命名文件
Delete - 删除选中文件
Ctrl+A - 选择所有文件
Ctrl+D - 复制文件
项目操作：
Ctrl+N - 新建项目
Ctrl+S - 保存项目
F5 - 刷新
Esc - 清除选择

  • □ 右键点击文件能显示上下文菜单
  • □ 双击文件能打开预览
  • □ 悬停文件能看到操作按钮
  • □ 能够重命名文件
  • □ 能够删除文件
  • □ 能够复制文件

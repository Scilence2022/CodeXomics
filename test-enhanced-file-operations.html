🔧 Project Manager Enhanced File Operations

🎯 实现的功能增强

  • 取消最小宽度限制: Project Manager 窗口可以自由调整到任意宽度
  • 双击文件打开: 双击文件树中的文件在主窗口 Genome AI Studio 中打开
  • 眼睛按钮预览: 点击 👁️ 按钮显示文件预览弹窗，而不是在主窗口打开
  • 单击选择不变: 单击文件仍然是选择功能

🔧 实施的代码修改

1. main.js 修改:
   - 移除了 minWidth: 1000 设置
   - Project Manager 窗口现在可以自由调整宽度

2. ProjectManagerWindow.js 修改:
   - 文件项添加了 ondblclick 事件处理
   - 眼睛按钮改为调用 showFilePreview() 方法
   - 实现了新的预览模态框系统
   - 添加了文件类型特定的预览内容

3. 文件操作行为:
   - 单击: selectFile() - 选择文件
   - 双击: openFileInMainWindow() - 在主窗口打开文件  
   - 眼睛按钮: showFilePreview() - 显示预览弹窗

📋 测试步骤

  1. 启动应用: 运行 npm start
  2. 打开 Project Manager: File → Project Manager
  3. 测试窗口调整: 尝试将窗口宽度调整到很小 ✅ 应该可以自由调整
  4. 测试文件操作:
    • 单击文件 → 选择文件
    • 双击文件 → 在主窗口打开
    • 点击 👁️ → 显示预览弹窗
  5. 验证预览功能: 预览弹窗应显示文件信息和内容示例

✨ 新预览功能特点

  • 美观的模态框界面设计
  • 显示文件详细信息（名称、类型、大小、修改时间）
  • 根据文件类型显示示例内容（FASTA、GFF、VCF、GenBank等）
  • "Open in Main Window" 按钮用于实际打开文件
  • 可通过多种方式关闭（X按钮、Close按钮、点击背景）

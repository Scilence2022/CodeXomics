Implementation Notes

  • GO Terms: Blue links that open to the AmiGO Gene Ontology database
  • PMIDs: Purple links that open to PubMed abstracts
  • Evidence Tags: Orange badges showing evidence codes (IBA, IEA, IDA, etc.)
  • Database Links: Green links to UniProt, InterPro, Pfam, KEGG, etc.
  • EC Numbers: Green links to the ENZYME database
  • Hover Effects: All links have smooth hover animations
  • Auto-detection: Regex patterns detect IDs in gene qualifiers automatically
  • External Links: All links open in new tabs for easy navigation

Usage Instructions

How to Access Debug Tools in GenomeExplorer:

  1. Open GenomeExplorer main window
  2. Click the Options button in the top toolbar
  3. Select Debug Tools from the dropdown menu
  4. Choose from the available debug tools in the modal
  5. Tools will open in new windows or iframe modals

Available Debug Tools:

  • BAM Query Debugging: test-bam-query-debugging.html
  • BAM Coordinate Fix: test-bam-coordinate-fix-validation.html
  • Performance Monitor: Built-in memory usage monitoring
  • Console Logger: Enhanced console logging with system state

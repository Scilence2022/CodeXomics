🔧 技术实现分析

函数共享机制: - 主窗口和Project Manager使用完全相同的窗口布局函数 - 所有布局函数定义在main.js的全局作用域中 - 通过getMainWindows()智能检测窗口实例 - BrowserWindow.getAllWindows()遍历所有窗口，根据标题识别 窗口检测逻辑: function getMainWindows() { const allWindows = BrowserWindow.getAllWindows(); const mainWindow = allWindows.find(win => win.getTitle().includes('Genome AI Studio') && !win.getTitle().includes('Project Manager') && !win.isDestroyed() ); const projectManagerWindow = allWindows.find(win => win.getTitle().includes('Project Manager') && !win.isDestroyed() ); return { mainWindow, projectManagerWindow }; } 菜单系统: - createMenu(): 主窗口菜单 - createProjectManagerMenu(): Project Manager菜单 - 两者的Window Layout子菜单完全一致 - 快捷键绑定保持统一（Ctrl+Alt+L/S/M/P）

API Integration Details

E-utilities APIs Used

// Search API https://eutils.ncbi.nlm.nih.gov/entrez/eutils/esearch.fcgi // Summary API https://eutils.ncbi.nlm.nih.gov/entrez/eutils/esummary.fcgi // Fetch API https://eutils.ncbi.nlm.nih.gov/entrez/eutils/efetch.fcgi

Search Parameters

db: Database name (pubmed, protein, nucleotide, etc.) term: Search query with field tags retmax: Maximum results per page (20-200) retstart: Starting position for pagination sort: Sorting options (relevance, date, author) retmode: Response format (json, xml)

Performance Metrics

Response Times

  • Search API: ~1-3 seconds
  • Summary API: ~2-5 seconds
  • Large datasets: ~5-10 seconds
  • UI rendering: ~100-500ms

Data Limits

  • Max results per request: 200
  • NCBI rate limit: 3 requests/second
  • Browser memory: ~10MB per 1000 results
  • Session timeout: 30 minutes

üìã User Workflow

Step 1: Switch to Edit Mode

Click the "Edit Mode" button in the sequence display area to switch from View Mode to Edit Mode with VSCode-style editor.

Step 2: Enable Editing

Click the "Enable Editing" button to activate sequence editing capabilities. The interface will show editing toolbar and status bar.

Step 3: Edit Sequence

Click on any base to edit it. Use keyboard navigation, copy/paste, or type new bases. Changes are highlighted in real-time.

Step 4: Save Changes

Click "Save" to submit changes to the Action Management System. An action will be created with detailed change summary.

Step 5: Execute Actions

Open the Action List (Window ‚Üí Show Action List) to review and execute the sequence editing action.
